# FitTrack

FitTrack es una aplicación web para el seguimiento de rutinas de gimnasio, entrenamientos y progreso físico.

## Tecnologías principales

### Frontend
- Vue 3
- TypeScript
- Vite
- Vue Router
- Pinia
- Composition API

### Backend
- PHP
- Laravel
- API REST

### Despliegue
- Docker
- Docker Compose
- NGINX

### Documentación
- MkDocs

## Documentación

La documentación del proyecto se gestiona con MkDocs. Para servirla en local:

```bash
mkdocs serve
```

Para generar el sitio estático:

```bash
mkdocs build
```
