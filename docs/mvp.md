# MVP

El Producto Mínimo Viable incluye las siguientes funcionalidades:

## Autenticación

- Registro de usuario
- Inicio de sesión
- Cierre de sesión

## Rutinas

- Crear rutinas
- Editar rutinas
- Eliminar rutinas
- Listar las rutinas del usuario

## Ejercicios

- Catálogo de ejercicios
- Filtro por grupo muscular

## Entrenamientos

- Iniciar sesión de entrenamiento
- Registrar series
- Almacenar repeticiones y peso

## Progreso

- Consultar historial de entrenamientos
- Estadísticas básicas de progreso

El objetivo del MVP es demostrar un flujo completo entre frontend, backend y base de datos.
