# Requisitos

## Requisitos funcionales

La aplicación debe permitir al usuario:

- Registrarse e iniciar sesión
- Crear y gestionar rutinas de entrenamiento
- Consultar el catálogo de ejercicios
- Registrar sesiones de entrenamiento
- Almacenar series, repeticiones y peso
- Consultar estadísticas básicas de progreso
- Gestionar su perfil

## Requisitos técnicos

**Frontend:**
- Vue 3
- TypeScript
- Componentes SFC
- Composition API
- Vue Router
- Pinia
- localStorage y sessionStorage

**Backend:**
- PHP
- Laravel
- API REST

**Despliegue:**
- Docker
- Docker Compose
- NGINX
